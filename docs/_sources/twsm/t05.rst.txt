5 轴向抗拔承载力
==============================

.. raw:: html

  <h2 id="test111">5 轴向抗拔承载力</h2>


5.1 一般规定
--------------------------------------  

.. raw:: html

 <p style="text-align:justify "><a href="#id5.1.4"> 5.1.4</a> <span id="id5.1.4"> 为了使轴向抗拔静载荷试验反映工程实际情况，要求试验桩成桩工艺和质量控制标准与工程桩一致。主筋应通长布置，以满足桩身强度的要求。</span></p>
 <p style="text-align:justify "><a href="#id5.1.5"> 5.1.5</a> <span id="id5.1.5"> 为设计提供依据时，应加载到能判别基桩抗拔极限承载力为止，或加载到桩身材料设计强度限值，工程桩验收检测时，要求加载量不低于基桩轴向抗拔承载力设计值2.0倍，旨在保证桩侧岩土阻力具有足够的安全储备。</span></p>
  <p style="text-align:justify;text-indent:2em;" > 桩侧岩土阻力的抗力分项系数比桩身混凝土要大、比钢材要大很多，因此时常出现设计对抗拔桩有裂缝控制要求时，抗裂验算给出的荷载可能小于或远小于基桩轴向抗拔承载力设计值的2.0倍，因此试验时的最大上拔荷载只能按设计要求确定。</p>




5.3 锚杆嵌岩桩的锚杆抗拔静载荷试验
--------------------------------------  

.. raw:: html

 
 <p style="text-align:justify "><a href="#id5.3.6"> 5.3.6</a> <span id="id5.3.6"> 锚杆验证性试验加载限制在抗拔力设计值的1.1倍～1.2倍，目的是保证工程桩不发生破坏，处于安全状态。</span></p>
